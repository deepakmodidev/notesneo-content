# NotesNeo Content Repository

📚 Educational notes content for the [NotesNeo](https://github.com/decodewithdeepak/notesneo) platform.

## 📂 Structure

```
notesneo-content/
├── 6th-sem/
│   ├── adbms/      # Advanced Database Management Systems
│   ├── adv-java/   # Advanced Java
│   ├── ai/         # Artificial Intelligence
│   ├── cd/         # Compiler Design
│   ├── ds/         # Distributed Systems
│   └── mwc/        # Mobile and Wireless Computing
└── 7th-sem/
    ├── fom/        # Fundamentals of Management
    ├── nn/         # Neural Networks
    ├── nsc/        # Network Security and Cryptography
    └── webm/       # Web Mining
```

## 📝 Content Format

Each subject contains:
- `unit-1.md` to `unit-4.md` - Unit-wise notes
- `syllabus.md` - Complete syllabus
- `pyq.md` - Previous year questions

### Frontmatter Format

```yaml
---
title: "Unit Title"
description: "Brief description"
author: "Author Name"
date: "YYYY-MM-DD"
pdfUrl: "Optional PDF download link"
---
```
